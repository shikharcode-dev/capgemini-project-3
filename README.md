# 🎮 Stone Paper Scissors Game (Python)

## 📌 Project Description

This is a menu-driven Stone Paper Scissors game where a user plays against the computer.
The computer generates random choices using Python’s random module.

---

## 🚀 Features

* 🎲 Random computer choice
* 🧠 Rule-based winner logic
* 🔁 Replay option
* 📋 Menu-driven system

---

## 🧠 Concepts Used

* Random Module (`random.randint`)
* Conditional Statements (if-elif-else)
* Dictionary (for mapping choices)
* Loops (while True)
* Functions

---

## 📂 Project Structure

game.py
main.py

---

## ▶️ How to Run

1. Open terminal
2. Navigate to project folder
3. Run:

```bash
python main.py
```

---

## 🧪 Example Workflow

1. Choose option to play
2. Enter choice (Stone/Paper/Scissors)
3. Computer generates random choice
4. Result is displayed
5. Option to play again

---

## 🎯 Key Highlight

* Used random module to simulate computer
* Implemented game logic using conditions
* Improved user experience with replay option
